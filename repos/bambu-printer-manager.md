# Rules: synman/bambu-printer-manager

Rules specific to the **synman/bambu-printer-manager** repository.  These rules extend and, where noted, override the global rules in `GLOBAL.md`.

## 1. Always confirm the correct repo before creating PRs

Before opening any pull request, explicitly confirm with the user that **synman/bambu-printer-manager** is the intended target repository.  This check is mandatory — do not skip it even when context seems obvious.

## 2. Do not open PRs for GitHub plan or platform-limit questions

Questions about GitHub Free/Pro limits, Actions minutes, storage quotas, or similar platform topics should be answered directly in chat.  Opening a PR to capture that information is not appropriate for this repository.

## 3. Accidental PR cleanup procedure

If a pull request is opened in this repository by mistake, follow these steps:

1. Close the pull request without merging.
2. Delete the associated head branch (both remotely and locally if checked out).
3. Note the PR number and a one-line description of what went wrong in the chat so the user has a record.
4. Confirm with the user that cleanup is complete before continuing.
